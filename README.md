# Jason AI Fintech Dashboard

Production-ready Next.js App Router dashboard with:

- AI chat interface backed by an API route
- Recharts-based analytics visualizations
- Realtime alert stream simulation
- Cinematic motion with Framer Motion
- Responsive Tailwind UI and reusable components

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` - start development server
- `npm run build` - production build
- `npm run start` - start production server
- `npm run lint` - run lint checks
